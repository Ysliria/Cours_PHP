Les opérateurs arithmétiques

Notions élémentaires

Les opérateurs arithmétiques fonctionnent exactement comme ils nous ont été appris à l'école.

  • Identité : converti une variable vers un integer ou un float, en fonction de ce qui est le plus approprié.
  • Négation : opposé de la valeur de la variable.
  • Addition : somme des valeurs des variables.
  • Soustraction : difference des valeurs des variables.
  • Multiplication : produit des valeurs des variables.
  • Division : quotient des valeurs des variables.
  • Modulus : reste de la division des valeurs des variables.
  • Exponation : élévaion de la première variable à la puissance de la seconde.

En quelques exemples : 

<?php
$maVariable = '3.67';
$monAutreVariable = 2;

+$maVariable; // Devient un float 3.67 et non une string
-$maVariable; // Devient -3.67

$maVariable + $monAutreVariable; // retourne 5.67
$maVariable - $monAutreVariable; // retourne 1.67
$maVariable * $monAutreVariable; // retourne 7.34
$maVariable / $monAutreVariable; // retourne 1.835
$maVariable % $monAutreVariable; // retourne 1
$maVariable ** $monAutreVariable; // retourne 13.4689

A noter qu'on peut effectuer l'affectation en même temps que l'opération arithmétique ; ainsi les 2 expressions suivantes sont equivalentes : 

<?php
$maVariable = $maVariable + 2;
$maVariable += 2;
