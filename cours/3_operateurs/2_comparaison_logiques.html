Les opérateurs de comparaison et les opérateurs logiques

Algèbre de Boole

L'algèbre de boole et la partie des mathématiques qui s'intéresse à la logique. Il exprime un état en fonction de conditions.
Cet état peut prendre deux valeurs : vrai ou faux. Ce résultat est déterminé en fonction des conditions et des opérateurs logiques utilisés ; il en resulte une table de vérité.
Les opérateurs (ou fonctions) logiques sont :

  • Conjonction, aussi appelée produit et représentée par le AND ; est vrai si et seulement si les 2 conditions sont vérifiées.
  • Disjonction, aussi appelée somme et représentée par le OR ; est vrai si au moins l'une des 2 conditions est vérifiée.
  • Négation, aussi appelée inverse et représentée par le NOT ; est l'inverse de la condition.
  • Disjonction exclusive, représentée par le XOR ; est vrai seulement si l'une des 2 conditions est verifiée, et pas les 2 !
  • Equivalence, représentée par le XNOR ; est vrai si les 2 conditions ont la même valeur.

Les opérateurs logiques

Ces fonctions ont des opérateurs en PHP.

  • Conjonction :
    • l'opérateur AND retourne TRUE si les deux variables valent TRUE $a AND $b;
    • l'opérateur && équivalent au précédent mais avec une priorité différente $a && $b;
  • Disjonction :
    • l'opérateur OR retourne TRUE si l'une des deux variables vaut TRUE $a OR $b;
    • l'opérateur OR équivalent au précédent mais avec une priorité différente $a || $b;
  • Disjonction exclusive : l'opérateur XOR retourne TRUE si l'une des deux variables vaut TRUE, mais pas les 2 $a XOR $b;
  • Négation : avec ! il retourne l'inverse de a !$a;

ATTENTION : la priorité des opérateurs peut vous donner des comportements differents ! Soyez vigilants !

Les opérateurs de comparaison

Ces fonctions ont des opérateurs en PHP.

  • Egal : on utilise l'opérateur == qui retourne vrai si les 2 valeurs sont égales $a == $b;
  • Identique : on utilise l'opérateur === qui retourne vrai si les 2 valeurs sont égales et que leur type est identique $a === $b;
  • Different :
    • on utilise l'opérateur != qui retourne vrai si les 2 valeurs sont différentes $a != $b;
    • on utilise aussi l'opérateur <> qui aura le même effet $a <> $b;
    • pour retourner vrai si les 2 valeurs sont différentes ou que leur type est différent, on utilise !== comme ceci $a !== $b;
  • Plus petit : on utilise l'opérateur < qui retourne vrai si a est plus petit que b $a < $b;
  • Plus grand : on utilise l'opérateur > qui retourne vrai si a est plus grand que b $a > $b;
  • Inférieur ou égal : on utilise l'opérateur <= qui retourne vrai si a est inférieur ou égal à b $a <= $b;
  • Supérieur ou égal : on utilise l'opérateur >= qui retourne vrai si a est supérieur ou égal à b $a >= $b;
  • Combiné : appelé aussi spaceship operator ou tie operator a cause de sa représentaion <=>, il retourne dans ce cas $a <=> $b;
    • -1 si a est inférieur à b
    • 0 si a est égal à b
    • 1 si a est supérieur à b

Exercice

Pour votre personnage, afficher le résultat suivant :

  • Ses points de vie sont ils inférieurs ou égaux à 200 ?
  • Ses points de vie sont ils identiques à '200' ?
  • Son nom est-il Mjolnir ?
  • L'inverse de ses points de vie.

Correction 

<?php
$merlin = [
    'nom'        => 'Merlin',
    'profession' => 'Sorcier',
    'pv'         => 200
];

var_dump($merlin['pv'] <= 200); // true
var_dump($merlin['pv'] === '200'); // false
var_dump($merlin['nom'] === 'Mjolnir'); // false
var_dump(!$merlin['pv']); // false : ce n'est pas 0 donc c'est vrai, l'inverse est donc false
