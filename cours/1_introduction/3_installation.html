Installation

Windows

Pour une installation sous Windows, il n'y a pas de grandes difficultés. Plusieurs solutions sont disponibles mais les plus simple restent l'installation des environnements complets.

WAMP

L'installation de WAMP est simple mais demande quelques attentions particulières.

  1. Aller sur le site de WAMP et télécharger la version souhaitée
  2. Avant de lancer l'installation il faut installer les packages redistribuables VC++ de Microsoft ; on peut trouver le lien ici ainsi que divers versions des constituants de WAMP pouvant être installé.
  3. Lancer l'installation de WAMP
  4. Une fois l'installation terminée, il faut valider que WAMP démarre bien et que son logo est bien vert

Laragon

Pour Laragon il suffit de lancer l'exécutable téléchargé sur le site officiel.

Contrôler l'installation de PHP

Chacune des solutions intègrent l'installation de PHP, pour valider son installation il suffit de lancer en ligne de commande php -v

MAC

Tout pourrait sembler aussi simple sur MAC, mais Apple a eu la mauvaise idée d'intégrer une version de PHP dans son coeur qui n'est pas mise à jour par leur soin !

Dans un premier temps installer MAMP.

ATTENTION ! Ne jamais installer MAMP sur Windows.

Une fois MAMP installé sur votre MAC, tenter de lancer la commande php -v.
Vous vous rendrez-compte que la version exécutée en ligne de commande et celle disponible dans MAMP ne sont pas les mêmes. Ce n'est pas bien, cela nous posera des problèmes. Il faut modifier la version exécutée par le MAC pour être en 7.4 !

  1. Installez Homebrew via la commande /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)".
  2. Puis, grace à Homebrew, installez la version de 7.4 de PHP avec la commande brew install php@7.4.
  3. Détacher la version actuelle de PHP du path du MAC avec la commande brew unlink php@X.X où les X correspondent à la version trouvée avec la commande php -v
  4. Ensuite liez la nouvelle version de PHP à votre console avec brew link php@7.4
  5. Redémarez la console et contrôlez la verion de PHP installée

X-Debug

Afin de nous faciliter la vie lors de nos exercices et de nos débogues nous allon installer X-Debug.
Pour ce faire :

  • affichez la page php_info() en fonction de votre environnement. Cela peut être fait en ligne de commande avec un php -i.
  • Copiez l'intégralité de son contenu.
  • Collez-le sur le site dédié.
  • Vous pouvez maintenant suivre la procédure d'installation qui est dédiée à votre environnement.
